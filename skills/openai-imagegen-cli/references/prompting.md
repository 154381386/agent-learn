# Prompting Guide

Use a compact structured prompt when the user's wording is vague.

## Template

```text
用途：<海报 / 封面 / 插画 / 产品图 / 头像 / banner>
主体：<主要对象>
风格：<写实 / 插画 / 3D / 扁平 / 复古 / 电影感>
构图：<近景 / 中景 / 远景 / 居中 / 留白 / 横版 / 竖版>
光线与氛围：<柔光 / 逆光 / 暖色 / 冷色 / 神秘 / 明亮>
细节：<材质、动作、背景元素、镜头语言>
文字：<如果有，写精确文案>
约束：<不要什么，必须保留什么>
```

## Short examples

### Product hero

```text
用途：网站 hero 图
主体：一只磨砂银色保温杯
风格：高质感商业产品摄影
构图：横版，主体居中，右侧留白
光线与氛围：柔和棚拍光，干净克制
细节：浅灰背景，杯身有细微金属反射
约束：不要 logo，不要额外杂物，不要水印
```

### Illustration cover

```text
用途：文章封面
主体：夜晚城市天台上看书的女孩
风格：日系插画，细节丰富
构图：中景，竖版
光线与氛围：深蓝夜色，暖黄色灯光
细节：风吹动头发和书页，远处城市灯火虚化
约束：不要文字，不要边框，不要水印
```

### Transparent sticker

```text
用途：贴纸
主体：抱着电脑的柴犬
风格：Q版卡通
构图：主体完整，独立站立
光线与氛围：明快、活泼
细节：轮廓清晰，适合裁切
约束：透明背景，不要阴影底板，不要文字
```
